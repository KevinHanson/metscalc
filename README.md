Metabolic Syndrome Severity Calculator
======================================

Calculates someone's Metabolic Syndrome Severity Score.

 * Website: https://metscalc.github.io/
 * Source code: https://github.com/metscalc/metscalc


Quickstart
----------

*Browser*

    $ make
    $ open www/index.html

*Command Line*

    $ node src/cli.js


Running Tests
-------------

    $ make test
